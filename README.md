# RestroomRater

Get Yelp API key from [Yelp Fusion](https://www.yelp.com/fusion) and set ```YELP_API_KEY``` environment variable.

### Installation
```
pip install requirements.txt
```

### Run
```
python manage.py runserver
```

in browser
```
127.0.0.1:8000/
```
